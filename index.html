This website exhibits CSS styles for pre-defined HTML tags that are used in @pr4j3sh/frames templates using Tailwind CSS.

Use these styles in your website using curl

curl -o style.css https://raw.githubusercontent.com/pr4j3sh/ui/refs/heads/master/style.css

Or, use wget

wget -O style.css https://raw.githubusercontent.com/pr4j3sh/ui/refs/heads/master/style.css

Include style.css in the head tag of your index.html file 

<link rel="stylesheet" type="text/css" href="./style.css">

Or, you can try one of frames' templates.
